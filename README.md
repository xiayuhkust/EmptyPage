# EmptyPage

A simple web application with MetaMask wallet connection functionality, built following Uniswap's architecture.

## Features

- MetaMask wallet connection
- Display of connected wallet address
- Error handling and loading states
- Built with React, TypeScript, and wagmi

## Development

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

## Architecture

This project follows Uniswap's wallet connection architecture, using:
- wagmi for Ethereum interactions
- React for UI components
- TypeScript for type safety
- Tailwind CSS for styling
